## 1. 요약

SendFeeCompare.com은 미국과 한국 사이에서 송금할 때 가장 좋은 환율과 가장 낮은 수수료를 찾을 수 있도록 도와주는 송금 비교 플랫폼입니다. 여러 사업자의 실시간 환율을 자동으로 스크래핑해 수수료, 환율 마진, 진행 중인 프로모션까지 포함한 총비용을 한눈에 비교해 보여 줍니다.

- 프로젝트 후기(한국어): [미국-한국 해외 송금 수수료 비교 사이트 제작기](https://www.milemoa.com/bbs/board/12093281)

### 핵심 가치 제안

- **환율 비교**: 사업자별 환율을 매시간 스크래핑
- **비용 분해**: 수수료·환율·총비용을 명확한 형식으로 표시
- **프로모션 트래킹**: 진행 중인 프로모션과 할인 표시
- **양방향 지원**: 미국→한국, 한국→미국 송금 모두 지원
- **다중 통화 표시**: USD 와 KRW 양쪽으로 비용을 보여 줘 직관적
- **다국어 지원**: 한국어/영어 완전 지원
- **모바일 반응형 디자인**: 데스크톱과 모바일 모두 최적화

### 비전

가능한 모든 옵션의 환율을 비교해 줌으로써, 개인이 국제 송금에서 정보 기반의 합리적인 결정을 내릴 수 있도록 돕고, 매번의 송금에서 시간과 돈을 아끼게 합니다.

## 2. 타깃 사용자

### 주요 사용자: "재미 한인 송금 사용자"

**목표:**
- 송금마다 가장 낮은 총비용 찾기
- 수취인이 정확히 얼마를 받게 되는지 이해
- 진행 중인 프로모션 정보를 놓치지 않기
- 빠르고 자신 있게 결정 내리기
- 정기 송금에서 비용 절감

## 3. 핵심 기능

### 3.1 구현된 기능

| 기능                       | 설명                                            |
| -------------------------- | ----------------------------------------------- |
| **사업자 비교**            | 9개 이상의 송금 사업자를 나란히 비교            |
| **수수료 분해 모달**       | 모든 비용 항목을 상세하게 분해                  |
| **프로모션 표시**          | 날짜 기반 필터링이 적용된 진행 중 프로모션      |
| **양방향 지원**            | 미국→한국, 한국→미국 송금                       |
| **언어 토글**              | 한국어/영어 인터페이스                          |
| **반응형 디자인**          | 모바일·데스크톱 최적화                          |
| **사업자 링크**            | 사업자 웹사이트로의 직접 링크                   |
| **장애 알림**              | 스크래핑 실패 시 Slack 알림                     |

### 3.2 금액 구간

**미국 → 한국 (USD):**
- $100, $500, $1,000, $2,000, $5,000, $10,000

**한국 → 미국 (KRW):**
- ₩100,000, ₩500,000, ₩1,000,000, ₩5,000,000

## 4. 기술 스택

### 4.1 프론트엔드

| 기술              | 버전     | 용도                                              |
| ----------------- | -------- | ------------------------------------------------- |
| **Next.js**       | 16.x     | 풀스택 React 프레임워크 (App Router)              |
| **React**         | 19.x     | UI 라이브러리                                     |
| **TypeScript**    | 5.9.x    | 타입 안전성                                       |
| **Tailwind CSS**  | 4.x      | 유틸리티 우선 CSS 프레임워크                      |
| **Radix UI**      | 최신     | 접근성 좋은 UI 프리미티브 (Dialog, Select)        |
| **Lucide React**  | 최신     | 아이콘 라이브러리                                 |
| **Zod**           | 4.x      | 스키마 검증                                       |

**프레임워크 버전 선택 근거:**

- **Next.js 16 with App Router**: 향상된 성능, 서버 컴포넌트, 더 나은 캐싱을 갖춘 최신 안정 버전
- **React 19**: 서버 컴포넌트 지원이 강화된 최신 기능
- **Tailwind CSS v4**: 큰 폭의 성능 개선과 새로운 기능
